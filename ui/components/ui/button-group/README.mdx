## Usage

The following describes the props and example usage for this component.

| Prop Type                  | Description                                                                         |
| -------------------------- | ----------------------------------------------------------------------------------- |
| `defaultActiveButtonIndex` | change button active order                                                          |
| `noButtonActiveByDefault`  | no buttons are active before clicked by the user                                     |
| `disabled`                 | disable every button inside button group                                          |
| `className`                | classname is used for children prop, changing it in storybook won't change anything |
| `style`                    | add style for button group component                                             |
| `newActiveButtonIndex`     | update value of active button in button group component                           |
| `variant`                  | options for rendering types of buttons, consists of 'default' and 'radiogroup'         |

### With Disabled Button

By changing the `disabled` value to true, the buttons inside the button group component will be disabled

```html
<ButtonGroup disabled>
  <button>cheap</button>
  <button>average</button>
  <button>fast</button>
</ButtonGroup>
```
